Documentation

ResearchStack has two primary modules. Backbone contains the core ResearchStack functionality and is designed to follow the APIs laid out in Apple's ResearchKit core as closely as possible. Skin is an application framework that implements the functionality of ResearchKit's AppCore. Links to the Javadocs for both Backbone and Skin are below; be aware that the documentation is itself a work in progress.
